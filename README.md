# Find Jobs API

🚧 This project is currently under development.

Find Jobs API is a RESTful backend service built with Laravel, designed to power a full-stack job portal application.

It will support:

- User authentication
- Job posting and management
- Applications system
- Company profiles
- Admin analytics

## Tech Stack

- Laravel (API backend)
- MySQL (Database)
- Sanctum (Authentication)
- Eloquent ORM

## Status

Currently building core backend features.

---

This project is being developed as a production-level SaaS application.
